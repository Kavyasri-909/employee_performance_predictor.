# 👨‍💼 Employee Performance Predictor

## 🚀 Overview
The **Employee Performance Predictor** is a Machine Learning project that predicts an employee’s future performance category (**High / Medium / Low**) based on key workplace factors such as experience, training, attendance, and workload.

This project simulates a real-world HR analytics system using **synthetic data** and provides an **interactive Streamlit dashboard** for predictions and insights.

---

## 🎯 Problem Statement
Organizations often struggle to:
- Identify high-performing employees
- Detect low performance early
- Make data-driven promotion decisions
- Plan employee training effectively

This project solves these problems using **Machine Learning-based prediction**.

---

## 💡 Solution
We built a system that:
- Generates realistic employee data
- Trains a classification model
- Predicts employee performance
- Visualizes insights through a dashboard

---

## 🧠 Machine Learning Approach

- **Problem Type:** Multi-class Classification  
- **Model Used:** Random Forest Classifier  
- **Features:**
  - Age
  - Experience
  - Salary
  - Training Hours
  - Projects Count
  - Attendance  

- **Target:**
  - Performance (High / Medium / Low)
